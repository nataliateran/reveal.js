Project Overview

The Verbecc/a> repository is a repository about a software that takes verbs from different foreign languages and predicts their conjugates in a desired tense. This project supports Catalan, Spanish, French, Italian, Portuguese and Romanian. This project can be used by anyone learning a lamguage or anyone who is fluent in a language, but is just stuck on how to conjugate something.
